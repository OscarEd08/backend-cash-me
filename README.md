
## Descripción

Este es el repositorio para el backend del proyecto Cash Me.

## Instrucciones

A continuación se detallan las instrucciones para ejecutar los contenedores Docker de este proyecto.

### Instalación de Docker

Antes de ejecutar los contenedores, asegurar de tener Docker instalado en el sistema. Si aún no está instalado, seguir estos pasos:

#### Paso 1: Instalar Docker

```bash
sudo apt update
sudo apt install docker.io
```

#### Paso 2: Verificar la instalación

```bash
docker --version
```

Este comando debería mostrar la versión de Docker instalada en su sistema.

## Ejecutar la aplicación

```bash
docker-compose up
```
